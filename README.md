# Logic Chatbot

A web-based chatbot that validates arguments using Rules of Inference, Rules of Replacement, and Categorical Propositions, and identifies the mood of categorical syllogisms (e.g., AAA-1, AAI-2). Provides step-by-step proofs for valid arguments and explanations for invalid ones.

---

## Features

- Validate propositional logic arguments:
  - Modus Ponens, Modus Tollens, Hypothetical Syllogism, Disjunctive Syllogism, etc.
- Apply rules of replacement:
  - De Morgan, Commutation, Association, Distribution, etc.
- Evaluate categorical syllogisms and output mood
- Strict JSON responses with verdict, proof, mood, and explanation
